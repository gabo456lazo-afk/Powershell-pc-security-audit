# Notas de aprendizaje

Este documento contiene apuntes personales sobre GitHub, PowerShell y ciberseguridad defensiva.

## Temas aprendidos

* Creación de repositorios en GitHub.
* Organización básica de un proyecto.
* Uso de archivos README.md.
* Uso de .gitignore para evitar subir archivos sensibles.
* Creación de scripts PowerShell para análisis defensivo.

## Buenas prácticas

* No subir información sensible a repositorios públicos.
* No publicar reportes reales de equipos personales o de terceros.
* Documentar cada script con su objetivo, uso y advertencias.
* Usar nombres claros para archivos y carpetas.

## Próximos temas por estudiar

* Comandos básicos de Git.
* Uso de GitHub Desktop.
* Firmas de commits.
* Markdown para documentación profesional.
* PowerShell aplicado a ciberseguridad defensiva.
